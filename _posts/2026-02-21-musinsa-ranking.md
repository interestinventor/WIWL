# 데이터 입력 필요

현재 `{{rankcrawler.products}}` 변수에 100개 상품 데이터가 입력되지 않았습니다.

아래 형식으로 데이터를 제공해주시면 즉시 Jekyll 포스팅을 작성해드리겠습니다:

```json
[
  {
    "rank": 1,
    "brand": "브랜드명",
    "product": "상품명",
    "price": 원가,
    "discount": 할인율,
    "final_price": 최종가격,
    "imageUrl": "이미지 URL",
    "link": "상품 링크"
  },
  ...
]
```

또는 실제 크롤링된 데이터를 직접 입력해주세요.
