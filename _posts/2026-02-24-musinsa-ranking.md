# 데이터 입력 필요

현재 `{{rankcrawler.products}}` 변수에 실제 무신사 랭킹 데이터가 입력되지 않았습니다.

아래 형식으로 100개 상품 데이터를 제공해주시면, 요청하신 Jekyll 블로그 포스팅을 즉시 작성해드리겠습니다:

```json
[
  {
    "rank": 1,
    "brand": "브랜드명",
    "product_name": "상품명",
    "price": 원가,
    "discount_rate": 할인율,
    "final_price": 최종가격,
    "image_url": "이미지 URL",
    "link": "상품 링크"
  },
  ...
]
```

또는 다른 형식의 데이터를 가지고 계시다면 그대로 입력해주세요. 데이터 형식에 맞춰 파싱 후 포스팅을 생성하겠습니다.
