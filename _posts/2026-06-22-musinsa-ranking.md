# 데이터 입력 필요

현재 `{{rankcrawler.products}}` 데이터가 제공되지 않았습니다. 

100개 상품의 데이터를 다음 형식으로 제공해주시면 즉시 Jekyll 포스팅을 작성해드리겠습니다:

```json
[
  {
    "rank": 1,
    "brand": "브랜드명",
    "product": "상품명",
    "price": 원가,
    "discount": 할인율,
    "finalPrice": 최종가격,
    "imageUrl": "이미지URL",
    "link": "상품링크"
  },
  ...
]
```

또는 CSV, 텍스트 등 어떤 형식이든 제공해주시면 분석 후 작성하겠습니다.
