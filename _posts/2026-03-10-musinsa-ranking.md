# 데이터 입력 필요

현재 {{rankcrawler.products}} 변수에 실제 무신사 랭킹 데이터가 입력되지 않았습니다.

다음 형식으로 100개 상품 데이터를 제공해주시면, 즉시 Jekyll 블로그 포스팅을 작성해드리겠습니다:

```json
[
  {
    "rank": 1,
    "brand": "브랜드명",
    "product": "상품명",
    "price": 원가격,
    "discount": 할인율,
    "final_price": 최종가격,
    "imageUrl": "이미지 URL",
    "link": "상품 링크"
  },
  ...
]
```

데이터를 입력해주시면 바로 사용 가능한 마크다운 포스팅을 생성하겠습니다.
