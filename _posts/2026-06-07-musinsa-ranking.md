# 데이터 입력 필요

현재 {{rankcrawler.products}} 데이터가 제공되지 않았습니다.

Jekyll 블로그 포스팅을 작성하려면 다음 형식의 데이터를 입력해주세요:

```json
[
  {
    "rank": 1,
    "brand": "브랜드명",
    "product": "상품명",
    "price": 원가,
    "finalPrice": 판매가,
    "discount": 할인율,
    "imageUrl": "이미지 URL",
    "link": "상품 링크"
  },
  ...
]
```

**또는** 무신사 크롤링 데이터를 텍스트/CSV/JSON 형태로 붙여넣어 주시면 즉시 WIWL 매거진 스타일의 포스팅을 작성해드리겠습니다.
