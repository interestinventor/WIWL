# 데이터 입력 필요

현재 `{{rankcrawler.products}}` 변수에 실제 데이터가 입력되지 않았습니다.

포스팅을 작성하려면 다음 형식의 데이터가 필요합니다:

```json
[
  {
    "rank": 1,
    "brand": "브랜드명",
    "product": "상품명",
    "price": 원가,
    "discount": 할인율,
    "final_price": 최종가격,
    "imageUrl": "이미지 URL",
    "link": "상품 링크"
  },
  ...
]
```

100개 상품의 실제 크롤링 데이터를 입력해주시면, 즉시 Jekyll 블로그 포스팅을 작성해드리겠습니다.
