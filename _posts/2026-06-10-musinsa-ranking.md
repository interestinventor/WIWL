# 데이터 미제공 안내

현재 `{{rankcrawler.products}}` 데이터가 입력되지 않아 포스팅을 작성할 수 없습니다.

**필요한 데이터 형식:**
```json
[
  {
    "rank": 1,
    "brand": "브랜드명",
    "product": "상품명",
    "price": 원가,
    "discountRate": 할인율,
    "finalPrice": 최종가격,
    "imageUrl": "이미지 URL",
    "link": "상품 링크"
  },
  ...
]
```

100개 상품 데이터를 제공해주시면 즉시 Jekyll 포스팅을 작성해드리겠습니다.
