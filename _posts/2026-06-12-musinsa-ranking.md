# 데이터 입력 필요

현재 `{{rankcrawler.products}}` 데이터가 제공되지 않았습니다. 

100개 상품의 다음 정보를 포함한 데이터를 입력해주세요:
- 브랜드명
- 상품명
- 가격
- 할인율
- 이미지 URL
- 상품 링크

데이터가 제공되면 즉시 Jekyll 블로그 포스팅 마크다운을 생성해드리겠습니다.

---

**데이터 형식 예시:**
```json
[
  {
    "rank": 1,
    "brand": "마르디 메크르디",
    "product": "후드 집업",
    "price": 89000,
    "discount": 20,
    "imageUrl": "https://...",
    "link": "https://..."
  },
  ...
]
```
